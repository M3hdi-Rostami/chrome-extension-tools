به‌روزرسانی اکستنشن

نسخه فعلی: —

برای بررسی نسخه جدید، دکمه زیر را بزنید.

مراحل تکمیل به‌روزرسانی:

  1. فایل ZIP در پوشه Downloads ذخیره شد.
  2. فایل را Extract کنید.
  3. محتوای پوشه داخل ZIP را جایگزین پوشه فعلی اکستنشن کنید.
  4. اگر با git نصب کرده‌اید: در همان پوشه git pull بزنید.
  5. در صفحه افزونه‌های Chrome روی Reload کلیک کنید.

تنظیمات ابزارها

ابزارهایی که نیاز ندارید را غیرفعال کنید تا در صفحه اصلی نمایش داده نشوند.
