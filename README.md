# chrome-extension-tools

آخرین نسخهٔ بیلد‌شدهٔ اکستنشن **Developer Widgets** (جعبه ابزار توسعه‌دهندگان).

## نصب در Chrome

1. آخرین نسخه را بگیرید:
   - **Clone:** `git clone <repository-url>`
   - یا **Download ZIP** از GitHub و Extract کنید
2. Chrome را باز کنید و به `chrome://extensions` بروید
3. **Developer mode** را فعال کنید
4. **Load unpacked** را بزنید
5. همین پوشه (`chrome-extension-tools`) را انتخاب کنید

## به‌روزرسانی

```bash
git pull
```

سپس در `chrome://extensions` روی دکمه **Reload** اکستنشن کلیک کنید.

## نکته

این پوشه خروجی بیلد است و به‌صورت خودکار از مخزن اصلی تولید می‌شود. برای توسعه، مخزن `super-extension` را clone کنید.
